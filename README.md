## 📋 Deskripsi Proyek
**Cusgo** adalah platform layanan penyewaan sepeda motor (Rental Motor) yang dirancang untuk memberikan kemudahan bagi pengguna dalam mencari dan memesan kendaraan secara daring. Fokus utama proyek ini adalah menghadirkan antarmuka pengguna (UI) yang modern, bersih, dan responsif, terinspirasi dari standar desain industri saat ini.

Proyek ini dikembangkan sebagai bagian dari mata kuliah **Desain Web** dengan penekanan pada penggunaan **HTML5**, **CSS3 (Box Model, Flexbox, & Grid)**.

<br />

## ✨ Fitur Utama
- [x] **Antarmuka Modern:** Desain minimalis berbasis tema *Slate/Dark Navy* (Shadcn-inspired).
- [x] **Katalog Armada:** Tampilan unit motor yang tertata rapi menggunakan sistem Grid/Flexbox.
- [x] **Formulir Kontak Interaktif:** Fitur pesan dengan validasi input dan desain form yang *clean*.
- [x] **Fully Responsive:** Tata letak yang beradaptasi secara otomatis untuk perangkat desktop, tablet, maupun ponsel.
- [x] **Struktur Navigasi Mulus:** Animasi garis bawah (*sliding underline*) pada menu untuk pengalaman navigasi yang lebih estetik.
